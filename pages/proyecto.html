Descripción de la propuesta del sitio (No más de 10 renglones)

Investigar para qué sirven los siguientes atributos y en qué etiquetas del sitio las podemos utilizar, incluirlas en el sitio del grupo

  1. target="_blank":

    Este atributo se utiliza en los elementos (hipervínculos) y se usa para especificar cómo se debe abrir el enlace cuando se hace clic en él. Cuando se establece target="_blank", el enlace se abrirá en una nueva ventana o pestaña del navegador. Por ejemplo: Visitar Ejemplo

  2. Pseudo-clase :hover:

    Esta pseudo-clase se utiliza en CSS para aplicar estilos a un elemento cuando el puntero del mouse se coloca sobre él. Es comúnmente utilizado en elementos interactivos como enlaces y botones para resaltarlos cuando el usuario pasa el mouse sobre ellos.
    a:hover { color: blue; text-decoration: underline; }

  3. overflow:

    Este atributo se utiliza en CSS para controlar cómo se debe comportar un elemento cuando su contenido excede su tamaño o dimensiones. Puede tener valores como hidden (ocultar el contenido que desborda), scroll (agregar barras de desplazamiento), auto (mostrar barras de desplazamiento solo cuando sea necesario), entre otros. Se utiliza comúnmente en elementos con dimensiones fijas, como cajas div. Ejemplo: .caja { width: 200px; height: 100px; overflow: scroll; }

  4. box-sizing:

    Este atributo se utiliza en CSS para controlar cómo se calculan las dimensiones de un elemento, incluyendo su contenido, relleno (padding) y borde (border). Puede tener valores como content-box (el tamaño total del elemento excluye el relleno y el borde) o border-box (el tamaño total del elemento incluye el relleno y el borde). Se utiliza para asegurarse de que los elementos se comporten de la manera esperada en diseño web. Ejemplo: .caja { width: 200px; height: 100px; padding: 10px; border: 2px solid black; box-sizing: border-box; }

  5. align:

    No es un atributo HTML o CSS estándar. Podría estar haciendo referencia a la propiedad CSS text-align, que se utiliza para alinear el texto dentro de un elemento, como un párrafo o un encabezado.
    p { text-align: center; }
